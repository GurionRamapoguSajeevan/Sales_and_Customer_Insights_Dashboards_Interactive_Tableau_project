# Key Features

# 1. Sales Performance Dashboard

## KPI Overview: *Total Sales*, *Total Profit*, and *Total Quantity* YoY comparison

- **Monthly Trends**: _Interactive monthly breakdown of KPIs for both current and previous years_
- **High/Low Indicators**: _Automatically highlights best and worst performing months_
- **Product Subcategory Comparison**: _Sales vs. Profit performance across categories
_
-----

# 2. Customer Insights Dashboard

## KPI Overview: *Total Customers*, *Sales per Customer*, *Number of Orders per Customer* (YoY)
- **Customer Trends**: _Monthly KPI trends + best/worst months highlighted_
- **Customer Order Distribution**: _Histogram-style analysis of ordering behavior & engagement_
- **Top 10 Customers by Profit**: _Ranking, Order counts, Sales & Profit, Last order date_
-----

# 3. Interactivity & Navigation

- Linked **Navigation Buttons** to switch between dashboards
- Dynamic **Filter** for _**Year, Product Category, Subcategory, and Geographic (Region, State, City)**_ Selector to explore historic data
- **Click-through Filtering** enabled on charts
